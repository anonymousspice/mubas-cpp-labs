# mubas-cpp-labs
A collection of MUBAS C++ programming lab exercises and projects completed during Year 1 at the Malawi University of Business and Applied Sciences. It covers fundamental programming concepts, including control structures, functions, arrays, pointers, object-oriented programming, templates, and file handling.
